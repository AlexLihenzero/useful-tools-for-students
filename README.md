## Notion

**Notion** is an all-in-one productivity and note-taking platform that helps students organize their academic and personal work. It combines notes, task management, databases, calendars, and collaboration tools in a single workspace, making it easier to keep track of assignments, projects, and study materials.

### Why Students Should Use It

* Organize class notes in one place.
* Create to-do lists and assignment trackers.
* Collaborate with classmates on group projects.
* Access notes from any device with cloud synchronization.
* Use templates for study schedules, project planning, and personal productivity.

### Key Features

1. **Note-Taking** – Create structured notes with headings, images, code blocks, and checklists.
2. **Task Management** – Track assignments, deadlines, and priorities.
3. **Databases** – Organize information using tables, boards, calendars, and lists.
4. **Collaboration** – Share pages and work together with classmates in real time.
5. **Templates** – Start quickly with pre-built templates for school, work, and personal use.

### How to Get Started

1. Visit https://www.notion.so/.
2. Create a free account using your email address.
3. Set up a workspace for your courses and projects.
4. Create pages for notes, assignments, and study plans.
5. Explore the template gallery to find layouts that suit your needs.

### Useful Links

* Notion Homepage: https://www.notion.so/
* Notion Help Center: https://www.notion.so/help
* Notion Templates: https://www.notion.so/templates

Notion is a valuable tool for students because it combines note-taking, organization, and project management into a single platform, helping users stay productive and manage their academic responsibilities efficiently.

## ChatGPT

**ChatGPT** is an AI-powered *conversational assistant* developed by OpenAI that helps students brainstorm ideas, explain complex biological concepts, and debug programming code in real-time. 

### Why Students Should Use It
- It breaks down dense topics into simple, digestible explanations.
- It can act as a 24/7 interactive tutor for subjects like Applied Biology and Python scripting.
- It helps automate repetitive drafting tasks, though students must always verify the accuracy of its output.

### How to Get Started
1. Create a free account at the official [OpenAI Website](https://chatgpt.com).
2. Type a clear, specific prompt in the chat bar (e.g., "Explain how translation works in molecular biology").
3. Review the response and ask follow-up questions to clarify specific details.

### Useful Links
- [ChatGPT Official Site](https://chatgpt.com)
- [Prompt Engineering Guide for Students](https://www.openai.com)
